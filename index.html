Experiments with the Two Stream radiative transfer model

ESA Summer School 2016

J Gómez-Dans (National Centre for Earth Observation and University College London)

This repository contains some experiments on using the Two Stream radiative transfer model to invert land surface parameters. The experiments are written in Python (2.7), and use IPython notebooks, and requires the installation of the eoldas_ng package, as well as the gp_emulators packages, as well as the standar Python scientific stack (numpy, scipy and matplotlib). You can probably install most of this using the Anaconda Python Distribution.

For more information, contact the author at j.gomez-dans AT ucl.ac.uk
